## 8.1 类的继承

### 补充知识：Java的向上转型和向下转型

定义：
- 父类引用指向子类对象为向上转型，语法格式为：<br>
  `superClass obj = new subClass();`
- 子类对象指向父类引用为向下转型，语法格式为：<br>
  `subClass obj = (subClass) superClass;`
